# Matrix Calculator

A professional web-based Matrix Calculator developed as a First-Year Engineering Mini Project. This project performs various matrix operations using a modern user interface built with HTML, CSS, and JavaScript while demonstrating matrix concepts commonly implemented using functions and 2D arrays in C programming.

## Project Overview

The Matrix Calculator is designed to perform essential matrix operations through an interactive graphical interface. Users can dynamically generate matrices of different sizes, enter matrix values, and perform mathematical operations with instant results.

The project focuses on understanding matrix manipulation, algorithm implementation, and frontend development concepts.

## Features

* Dynamic Matrix Generation
* Matrix Addition
* Matrix Multiplication
* Matrix Transpose
* Determinant Calculation
* Matrix Inverse Calculation
* Input Validation
* Animated Result Display
* Responsive User Interface
* Modern Glassmorphism Design
* Matrix Size Selection (2×2, 3×3, and 4×4)

## Supported Operations

### Matrix Addition

Adds two matrices of the same dimensions.

### Matrix Multiplication

Performs matrix multiplication between Matrix A and Matrix B.

### Matrix Transpose

Converts rows into columns and columns into rows.

### Determinant

Calculates the determinant of the selected matrix.

### Matrix Inverse

Computes the inverse of a matrix when it exists.

## How to Run the Project

1. Download or clone the repository.

2. Open the project folder.

3. Locate the file:

   index.html

4. Double-click the file or open it in any modern web browser.

5. Select the matrix size.

6. Generate matrices.

7. Enter matrix values.

8. Perform the desired matrix operation.

No additional software installation is required.
