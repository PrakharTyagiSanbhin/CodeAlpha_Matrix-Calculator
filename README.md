# Matrix Calculator

A professional Matrix Calculator developed as a First-Year Engineering Mini Project. It is built using **HTML, CSS, and JavaScript** for **Frontend** and **C programming** for **Backend implementation** using functions and two-dimensional arrays.

---

## Project Overview

The Matrix Calculator is designed to perform fundamental matrix operations such as:

* Matrix Addition
* Matrix Multiplication
* Matrix Transpose
* Determinant Calculation
* Matrix Inverse (Web Version)

The project provides an interactive graphical interface for users to perform matrix calculations while also demonstrating the backend implementation of matrix operations in C programming.

## How to Run the C Backend Program

### Using Code::Blocks

1. Open Code::Blocks.
2. Create a New Console Application.
3. Select C Language.
4. Replace the default source code with the contents of:

```text
backend c language.c
```

5. Save the file.
6. Press F9 to Build and Run.

## Supported Matrix Operations

### Matrix Addition

Adds corresponding elements of two matrices.

### Matrix Multiplication

Performs matrix multiplication using row-column multiplication rules.

### Matrix Transpose

Converts rows into columns and columns into rows.

### Determinant

Calculates the determinant of:

* 2×2 Matrices
* 3×3 Matrices

### Matrix Inverse

Available in the web application version.
